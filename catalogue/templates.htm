Description Templates

This page contains the XML description template and the XSLT to transform the XML files into HTML pages, which we used in the catalogue for the project.

We developed two levels of description and the stylesheets are contained in this architecture as follows:

  • Download XML Template
  • Download XSLT for Descriptions (underlined links in the list of manuscripts)
  • Download XSLT for Descriptions with a Physical description and History (underlined links with bold fonts in the list of manuscripts)

These materials are provided on an open source basis, but please acknowledge the project in your further work.

If your browser does not support the download function, right click and select "Save Target As...", this will allow you to use the files from your system.
