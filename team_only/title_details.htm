List of Titles, which we itemized further

When Cameron's Itemization is not detailed enough, we will further itemize them.

_extract

Any item, when it is an extract version of the full version. For example, See for example:

Vesp. D. xiv, Extract from Ælfric's Third Series of Homilies [Lives of Saints], Memory of the Saints (fols 22r/19 - 23v/20): xml:id="B.1.3.17_extract"

B.3.1.2

The Vercelli Homilies: in Cameron, Vercelli Homilies is not itemised further than this. So, we will put the Homily numbers in Roman Alphabet, following Scragg's edition. For example:

  • ID no: B.3.1.2.IV
  • How to refer: <hi rend="title" >The Vercelli Homilies</hi >, 'Homily IV'

B.8.4

B.9.1

Cameron says: '9. Alfredian and Other Translations', but when we know that it is Alfredian Translation, we can omit 'and Other'.

'Gregory the Great, The Pastoral Care' is a single item in Cameron, but we subdivide it into:

  • Preface (xml:id="B.9.1_pre")
  • Metrical Note (B.9.1_metrical)
  • Table of Contents (B.9.1_toc)
  • the main text, i.e., The Pastoral Care (B.9.1)

B.9.3

Cameron says: '9.3 Boethius, The Consolation of Philosophy'. We subdivide it into:

  • Preface (xml:id="B.9.3_pre")
  • Table of Contents (B.9.3_toc)
  • the main text, i.e.,Consolation of Philosophy (B.9.1)
